# सरल लेखन (Saral Lekhan)

This folder contains planning artifacts and the beginnings of a production-ready React Native app built with Expo.

## Getting started

1. Install dependencies: `npm install` or `yarn`.
2. Start Metro: `npm run start`.
3. The `src/` directory contains the application structure described in the design documentation.

## Architecture
See `implementation_plan.md.resolved` for a detailed plan, including token definitions, UI components, and screen workflows.
