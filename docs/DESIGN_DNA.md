# Design DNA - Saral Lekhan Plus 🎨

This document defines the core aesthetic and functional principles for Saral Lekhan Plus, drawing inspiration from premier Material 3 open-source applications.

## 🏛 Core Inspiration Sources
1.  **Material 3 (M3)**: The foundational design language (Adaptive colors, intentional whitespace, expressive typography).
2.  **Image Toolbox**: For its masterclass in **Dynamic Colors (Monet)** and extensive UI customization (shapes, sliders, glassmorphism).
3.  **Mihon**: For superior **Library Management** (Categories) and deep, well-organized settings.
4.  **Metrolist**: For clean, high-density listing patterns.

## 💎 Design Pillars
### 1. Dynamic Personalization (The "Monet" Effect)
- The app should feel native to the user's device by adopting system-wide color schemes.
- **Goal**: Implement image-based color extraction for custom themes.

### 2. Expressive Components
- Use "Bento Grid" layouts for data density without clutter.
- Incorporate smooth animations and transitions (Framer Motion style).
- Standardize on `radius.lg` (24px) for cards to match the M3 aesthetic.

### 3. Deep Customization
- Users should be able to tweak secondary UI elements (like card shapes or slider styles).
- Advanced font scaling beyond simple % values (Normalizing across different scripts).

### 4. Categorical Organization
- Hierarchical folders and smart filtering inspired by Mihon's library system.
