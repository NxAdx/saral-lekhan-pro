# Release v2.12.0: Startup & Theme Redesign

**Date**: 2026-03-06
**Version**: 2.12.0 (versionCode 44)

## Summary
Focus on premium startup experience, theme simplification, and updater logic hardening.

## Key Changes
- **⚡ Startup Polish**: Eliminated the 'white flash' during boot. The splash screen now persists until fonts and the database are ready. Redundant skeleton loaders removed.
- **🎨 Theme Simplification**: Removed legacy 'Amoled Mode' and the 'Pitch' theme. Dark modes are now natively high-contrast for better usability.
- **🛠️ Strict Updater Logic**: Switched to GitHub `/releases` API for robust version tracking. Strictly notifies only on newer versions (no more unexpected reinstall prompts).
- **📱 Themed UI Popups**: Replaced native `Alert` calls with `ThemedModal` for all update-related notifications to maintain a premium feel.
- **🧹 Garbage Cleanup**: Deleted unused `HomeSkeleton` and `HardShadow` components.

## Verification
- Verified persistence of splash screen (no white flash).
- Verified removal of Amoled/Pitch settings.
- Verified ThemedModal appears for manual update checks.
- Verified app version bumps in `app.json`.
