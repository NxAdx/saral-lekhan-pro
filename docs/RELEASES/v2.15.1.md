# Release Notes: v2.15.1

## Core Updates

### 🚀 Advanced Android Updater (Direct Install)
- **Native Integration**: Implemented a custom Android module leveraging the `PackageInstaller` Session API.
- **One-Click Experience**: Updates now stream directly into the system installer, providing a seamless "Direct Install" experience without secondary app intervention.
- **Expert Versioning**: Replaced string-based version comparison with a robust numeric weight logic (`major * 100M + minor * 10K + patch`) to ensure 100% accurate update detection.
- **OEM Compatibility**: Added automatic detection for Xiaomi (MIUI) devices with a resilient fallback mechanism to handle specialized restricted environments.

### 🌐 Localization & Stability
- **Critical Crash Fix**: Resolved a high-priority issue where switching to Bengali, Tamil, or Telugu locales caused the application to crash due to missing configuration keys.
- **Global Synchronization**: Fully synchronized all 6 supported languages. Every locale now has complete support for Feature Discovery, Typography settings, and Support links.
- **Aesthetic Cleanup**: Removed decorative emojis from all translation files to ensure a clean, professional, and consistent UI across all languages.
