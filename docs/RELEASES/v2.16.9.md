# Release Notes - v2.16.9

## Summary
- Kept the native splash on screen until the first real app layout to reduce the second splash and blank launch gap.
- Fixed intermittent editor gallery image failures by embedding picker Base64 as JPEG data before insertion.
- Routed editor text sizing and settings pills through the effective font scale for better font-switch consistency.
- Flattened the `Biometric Vault` card border treatment to remove the clipped shadow / double-border effect.
- Prepared this fix as a tagged GitHub release so the direct updater can install it.

## Verification Focus
- Cold launch on Android should no longer show two splash phases followed by a blank `#d9d7d2` frame.
- New gallery images inserted into both new and existing notes should render reliably after save and reopen.
- Switching among `Hind`, `Poppins`, `Noto Sans`, and `Baloo 2` should feel more even in editor text and settings pills.
- `Biometric Vault` should look like a single clean card, not a double-border card.
