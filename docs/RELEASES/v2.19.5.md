# Release Notes - v2.19.5

Date: 2026-06-26
Tag: `v2.19.5`

This release resolves the critical version sync mismatch issue that prevented the in-app updater from detecting updates, and consolidates the major selection mode fixes.

## What changed

### 1) Version Resolution and Configuration Sync
- **Duplicate Key Fix:** Removed a duplicate `"version": "2.19.1"` entry at the bottom of `package.json` which was overriding the target version string and causing `app.config.js` and the Gradle `JsonSlurper` compiler to statically bake version `2.19.1` (versionCode 105) into the production APK.
- **OTA Updater Compatibility:** By correctly publishing as version `2.19.5` (versionCode 106), the GitHub updater will now be able to resolve and verify updates properly when a newer release is published on GitHub.

### 2) Selection Mode UI and Responsiveness
- **Layout Restructuring:** Redesigned the selection header to use a responsive two-row layout on mobile viewports so that the "Delete" and "Export" buttons are prominent and never overflow off-screen.
- **Visual Selection Rings:** Added clear empty circular indicators to all unselected cards when entering selection mode, guiding users on tap targets.
- **State Deadlock Fix:** Resolved a critical React Hook state updater loop deadlock that caused the app to freeze when tapping cards or action buttons.
- **Rendering Performance:** Streamlined `FlashList` memoization and updated the `extraData` dependency key to force-refresh item views during selection.
