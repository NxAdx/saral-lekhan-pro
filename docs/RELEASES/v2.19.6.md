# Release Notes - v2.19.6

Date: 2026-06-26
Tag: `v2.19.6`

This release fixes potential header rendering and state synchronization lag inside FlatList/FlashList.

## What changed

### 1) Header Layout State De-caching
- **Removed ListHeader Caching:** Decoupled `ListHeader` from `useMemo` so that the custom toolbar (Delete, Export) and counts update seamlessly on every selection transition.
- **Improved Input Focus Retention:** The root layout is preserved correctly during standard typing operations in search, but now forces render checks when switching modes.

### 2) Direct OTA Updater Track
- Bumped app build targets to `2.19.6` (versionCode `107`).
