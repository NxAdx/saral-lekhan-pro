# Release Notes - v2.9.3 🚀

## Core Improvements

- **Marathi Font Fallback System:** Implemented a robust font-family pipeline in `src/styles/typography.ts`. Latin characters are still rendered beautifully using premium fonts, while Devanagari text (Marathi, Hindi) now gracefully falls back to optimized system fonts.
- **Google Drive Sync Engine Fix:** Hardened the SQLite Backup engine. The app now implicitly refreshes the OAuth token before any push/pull operation.
- **Production Build Architecture:** Resolved persistent "Major Version 64" errors by enforcing Java 17 compatibility across the entire Gradle task graph.
- **Offline Build Documentation:** Added a comprehensive guide in `docs/BUILD_OFFLINE.md` for local compilation.
