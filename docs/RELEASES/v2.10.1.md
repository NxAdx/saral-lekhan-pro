# Release v2.10.1 (March 4, 2026) - Production Final Polish

## ⚡ Instant Launch & Startup Optimization
- Database pre-loading moved from Home screen mount into `_layout.tsx` (concurrent with fonts/splash).
- App now renders Home screen instantly after splash — no more pulsing gray `HomeSkeleton`.
- Unified startup background to `#171513` to prevent white flash on cold boot.

## 🔄 Reliable Updater
- Refactored `githubUpdater.ts` to use fire-and-forget intents.
- Added `REQUEST_INSTALL_PACKAGES` permission in `app.json` (Android 11+ requirement).
- Fixed issue where the app hung at "Downloading 100%" instead of launching the installer.

## 🔐 Auth & Login Stability
- GitHub CI now prints both **SHA-1** and **SHA-256** fingerprints for copying to Firebase.
- Added strict `GOOGLE_SERVICES_JSON` validation in CI script.
- Improved error handling for `DEVELOPER_ERROR` (Error 10) on Play integrity mismatches.

## 📋 Changelog & Docs
- Updated in-app changelog with EN/HI/MR translations.
- Version bump: `v2.10.1` (versionCode 43).
- Full sync of master project documentation and error resolution logs.
