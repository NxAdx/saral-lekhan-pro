# Production Refinements (v2.8.0)

This update polishes the user interface and resolves critical issues with image rendering.

## 📍 Home Screen Enhancements
- **Top Bar Relocation**: The **"History"** (Changelog) and **"Report Bug"** icons are now readily accessible from the top bar of the Home screen.
- **Support Cleanup**: Redundant support sections have been removed from Settings to keep the interface clean.

## 🖼️ Editor Stability & Fixes
- **Image Rendering Fix**: Images picked from the gallery are now converted to Base64 data URIs. This ensures they render correctly within the editor and persist even if the original file is moved or deleted.
- **Tag Input Hints**: Added a helpful hint below the tag input field to guide new users.
- **PDF Support Removal**: As requested, PDF import support has been removed to focus on document editing.

## 🎨 UI & Accessibility
- **Themed Modal Fix**: Improved the contrast of button text within modals across all themes. Buttons are now clearly visible in both light and dark modes.

## 🚀 How to Verify
1.  **Check Home Header**: Verify the Clock (History) and Warning (Report Bug) icons appear next to the Settings icon.
2.  **Pick Image**: In the editor, insert an image from your library. It should appear immediately and stay visible.
3.  **Check Tags**: Notice the new hint text when editing a note's tag.
4.  **Try Export**: Open the export menu and verify all button labels are clearly legible.
