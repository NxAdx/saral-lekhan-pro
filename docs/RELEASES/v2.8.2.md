# Walkthrough - v2.8.2 Theme Contrast Fix

This point release addresses a critical contrast issue in the "Pitch" dark theme where modal button text was invisible.

## Key Fixes

### 🌗 Pitch Theme Visibility
- **Issue**: In the "Pitch" theme, the `white` token is mapped to black (`#000000`) for high-contrast "black-out" aesthetics. My previous fix used `colors.white` for dark-mode labels, resulting in black text on a black background in this specific theme.
- **Fix**: Switched the `ThemedModal` button labels to use the `ink` token. Since `ink` is correctly mapped to White (`#FFFFFF`) in Pitch-dark, the text is now perfectly visible.
