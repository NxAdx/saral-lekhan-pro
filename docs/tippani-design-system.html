Border Radius — Pill System

07. आकार

This app is pill-forward. Every interactive element defaults to --r-pill or --r-xl. Smaller radii (--r-md, --r-sm) are reserved for nested elements inside cards only.

--r-pill · 9999px Note pills, tags, inputs, FAB
--r-xl · 28px Hero, modal sheets, textareas
--r-md · 14px Toolbar keys, icon cells

Motion & Animation

08. गति

Motion in Tippani should feel mechanical and physical — like pressing a real key. The signature gesture is the press-down: translateY(2px) + shadow collapse on tap. Hover lifts, tap presses.

Key Press
translateY(2px)
Shadow collapses on tap. Simulates physical key depression. 80ms ease-in.
Pill Enter
ease-out · 220ms
Note pills slide in from bottom with stagger. 40ms between each card.
Spring Pop
spring(500, 40)
FAB appears with spring. Tag selection snaps with scale-pop micro-animation.
Swipe Delete
drag · x-axis
Swipe left on pill reveals delete. translateX + opacity to 0 on confirm.
Sheet Rise
spring · bottom sheet
Edit screen slides up from bottom. Backdrop fades to 40% black behind.
Auto-save Pulse
ease-in-out · 1.2s
Subtle opacity pulse on save indicator. Fades in "saved ✓" after 600ms.
